# 🚀 Sept Optimizer

Ferramenta avançada de otimização profunda para Windows, focada em desempenho real, estabilidade e controle total do sistema.

---

## 📥 Download

👉 **Baixe a versão mais recente:**

🔗 (Baixar)[https://github.com/armalunar/septoptimizer/releases/latest/download/SeptOptimizer1.5.exe]

---

## ⚡ Principais Recursos

- 🧠 Auto Tune inteligente (Perfil Geral e Gamer)
- 🧹 Limpeza profunda (cache, temporários, logs)
- 🎮 Otimizações para jogos (FPS Boost, estabilidade)
- 💾 Otimização de disco (TRIM / Desfragmentação)
- ⚙️ Tweaks de registro e kernel
- 🔌 Gerenciamento avançado de energia
- 📊 Logs completos de todas as ações
- 🔄 Suporte a reversão via backup

---

## 🧪 Como usar

1. Baixe o executável acima
2. Execute como **Administrador**
3. Escolha uma otimização:

### 🔹 Nível 1 (Seguro)
- Auto Tune (Recomendado)
- Deep Clean
- Disk Boost

### 🔹 Nível 2 (Intermediário)
- FPS Boost
- Boot Optimize
- System Repair

### 🔹 Nível 3 (Avançado)
- Kernel Tweaks
- Debloat Services
- Registry Tweaks

---

## 📂 Logs e Dados

Os arquivos são salvos em:
%AppData%\SeptOptimizer\

---

## ⚠️ Aviso

Este software realiza modificações profundas no sistema.

- Use por sua conta e risco
- Recomenda-se criar ponto de restauração
- Pode impactar serviços do Windows dependendo do uso
